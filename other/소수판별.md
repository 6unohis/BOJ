<h2>1. 2부터 N-1까지 나눠보기</h2>

- N이 소수인지 판별할 때, 2부터 N-1까지 나누어 떨어지는 수가 하나라도 있다면 소수가 아님.

<br/>

<h2>2. 에라토스테네스의 체</h2>

- 배열을 초기화하여 소수 상태값 저장.

- 1을 소수가 아닌 수, 0을 소수라고 했을 때
```c
for(i = 2; i <= n; i++){
    if(A[i]) continue;
    else{
        for(j = i + i; j <= n; j+=i) A[j] = 1;
    }
}
```
